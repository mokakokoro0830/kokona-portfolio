— Process

どうやって作るか

— From concept to launch, solo —

01

業務分析・要件整理

「何を繰り返しているか」を言語化。どこを自動化すると一番インパクトがあるかを整理する。

02

ツール選定・設計

Claude / GAS / API / Next.js の中から最適な組み合わせを選ぶ。作りすぎない設計を心がける。

03

実装・自動化

Claudeとの対話で高速に作る。フォーム・DB・メール・管理画面まで一気通貫で組む。

04

デプロイ・運用改善

Vercel / Cloudflare で即公開。動かしながら改善する。自分でも使い続けることが品質保証。

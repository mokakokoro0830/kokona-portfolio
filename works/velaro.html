RESORT · BOOKING SYSTEM

VELAROヴェラーロ

カテゴリ

リゾート / 予約管理システム

技術

Next.js / Supabase / Resend

デプロイ

Vercel

ステータス

LIVE

概要

架空のリゾートヴィラ「VELARO」向けの予約管理システム。単なるLPではなく、ゲストの予約から管理者の承認・メール送信までを一気通貫で動かすフルスタック実装です。

ゲストが予約フォームを送信すると確認メールが自動送信され、管理者は管理画面でステータス管理・任意のメール送信が可能。実業務に近い設計を意識しました。

こだわり

  • ゲスト向けUI（ミニマル）と管理画面（情報密度高め）でデザイン思想を分けた
  • 予約作成と同時にゲスト確認メール＋管理者通知メールを自動送信（Resend）
  • 管理画面でステータス変更 → 自動でゲストにメール通知
  • 完成後に自分でセキュリティレビューを実施、5件の問題を修正してデプロイ

技術構成

  • Next.js 16（App Router）— フロント＋API Routes
  • Supabase — 予約データDB・RLS設定
  • Resend — トランザクションメール送信
  • Tailwind CSS v4 — スタイリング
  • Vercel — デプロイ・環境変数管理

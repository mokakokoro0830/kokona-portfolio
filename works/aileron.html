概要

架空のAI業務自動化スタートアップ「Aileron Inc.（エイルロン）」のコーポレートサイト。WordPressオリジナルテーマでの実装を想定して、全13ページの構造を静的HTMLで再現しました。

クライアント業務でよくある「Excelで作られたページ構成を元に、既存サイトを参考にコンテンツを作成する」案件を意識し、コーポレートサイトに必要な情報設計・テンプレート階層・カスタム投稿（事例・お知らせ）の運用イメージを具体化しています。

ページ構成（全13ページ）

  • トップ（Hero / Services / Stats / Cases / News / CTA）
  • 会社案内（理念 / 代表メッセージ / 会社概要 / 沿革 / アクセス）
  • サービス一覧 + 詳細3ページ（DXコンサル / AI業務自動化 / システム開発）
  • 導入事例一覧 + 事例詳細サンプル（カスタム投稿想定）
  • お知らせ一覧 + お知らせ詳細サンプル（カスタム投稿想定）
  • 採用情報（職種別募集要項 / 働き方 / 選考フロー）
  • お問い合わせ（バリデーション付きフォーム / プライバシー同意）
  • プライバシーポリシー

こだわり

  • WordPress運用前提の構造設計：事例・お知らせはカスタム投稿タイプ想定、フィルタ・カテゴリ表示を意識
  • BEM準拠のクラス命名 + W3C準拠のセマンティックHTMLでSEO構造を担保
  • パンくず・OGP・meta descriptionを全ページに実装
  • レスポンシブ完全対応（375 / 768 / 1280 / 1920px で実機確認）
  • FAQアコーディオン・モバイルメニュー・スムーススクロールをjQueryなしで純粋JSで実装
  • IntersectionObserverで軽量なスクロール出現アニメーション

技術構成

  • HTML5（セマンティックタグ・microdata想定）
  • CSS3（Custom Properties・Grid・Flexbox）
  • Vanilla JavaScript（IntersectionObserver / FAQ Accordion / モバイルメニュー）
  • Google Fonts（Inter + Noto Sans JP）
  • Cloudflare Pages（デプロイ・無料SSL）
  • WordPress化を想定：カスタム投稿タイプ・ACF・functions.php 責務分離
